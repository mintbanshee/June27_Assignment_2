# 🌸 Foxglove Journal

## 📝 Description
**Foxglove Journal** is a digital sanctuary for recording your thoughts, reflections, and gentle daydreams. With a soft, cottagecore aesthetic, it provides a calm space to journal about your day, track your moods, and cherish the quiet moments of life.

---

## 🍃 Features
- 🌿 Write and edit personal journal entries
- 🌤️ Add mood and weather tags
- 📸 Upload images to accompany entries
- 🔐 Toggle privacy for sensitive entries
- 🌙 Clean, soft aesthetic inspired by cottagecore vibes

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla or framework TBD)
- (Optional) LocalStorage for saving entries

---

## 🐞 Known Bugs
- Image uploads may not preview correctly in some browsers
- Journal entries do not yet persist after page reload (no backend)
- Mood tag colors may not display properly in dark mode

---

## 🌷 Future Features
- User login system for saving and accessing entries
- Soundscape toggle for ambient journaling (e.g., rain, birdsong)
- Markdown support for journal formatting
- Daily cozy prompts or inspirational quotes
